## 数据库说明

### 默认使用 SQLite（推荐开发使用）

**无需安装任何数据库软件**，直接运行即可。

数据库文件位置：

```
D:\code\在线投票系统\
└── vote.db   ← 所有数据都保存在这个文件里
```

启动后 `vote.db` 会自动生成。你可以用以下工具可视化查看数据：
- [DB Browser for SQLite](https://sqlitebrowser.org/)（免费，推荐）
- VS Code 插件：SQLite Viewer

---

### 切换到 MySQL（生产部署时使用）

1. 安装 MySQL 驱动：
   ```bash
   pip install PyMySQL
   ```

2. 修改 `config.py`，将最后一行改为：
   ```python
   config = {
       ...
       'default': MySQLConfig   # 改用 MySQL
   }
   ```

3. 修改 `.env` 文件中的数据库连接信息：
   ```
   DATABASE_URL=mysql+pymysql://root:你的密码@localhost:3306/vote_db?charset=utf8mb4
   ```

4. 在 MySQL 中创建数据库：
   ```sql
   CREATE DATABASE vote_db CHARACTER SET utf8mb4;
   ```

---

### 两种数据库对比

|          | SQLite              | MySQL              |
|----------|---------------------|--------------------|
| 安装     | ✅ 无需安装         | ❌ 需要安装服务    |
| 数据位置 | 项目目录 `vote.db`  | MySQL 数据目录     |
| 适合场景 | 开发 / 学习 / 单机  | 生产 / 多人并发    |
| 性能     | 中等（够用）        | 高（推荐生产）     |
| 备份     | 直接复制 .db 文件   | 需用 mysqldump     |
